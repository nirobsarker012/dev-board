Question-1: What are the different ways to select on element in the DOM?

I can select an element in the DOM using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the different between innerHTML and textContent?

innerHTML returns or sets the HTML content within an element, including tags, while textContent returns or sets only the plain text content, ignoring HTML tags.

Question-3:What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element to handle events for all its child elements, leveraging event bubbling to manage dynamic or multiple elements efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates upward through its ancestor elements in the hierarchy, allowing parent elements to handle the event.
